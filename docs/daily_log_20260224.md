\### 3. 发现的 Bug



\*\*GPT-5.2 的 `\_\_ldg` 错误\*\*:

```cuda

// 错误代码

const double4 xi4 = \_\_ldg(\&x4\[i]);  // ❌ \_\_ldg 不支持 double4



// 正确代码

const double4 xi4 = x4\[i];  // ✅ 直接访问

```



原因: `\_\_ldg()` 只支持基本类型 (int, float, double)，不支持向量类型 (double4, float4)
